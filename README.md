# Re-Fine
A mod planned to be a successor to optifine. For now it is going to be fabric only, for ease of quick development, until release 1.0 where I will make forge, neoforge, and quilt versions.

Started development: 21/07/2025 (UK) 07/21/2025 (US)

## Planned Features

-  Vulkan based rendering
-  Use of RT cores for shaders and DLSS for performance boost
-  99% Optifine compatibiliy; all Optifine file types/configurations will be compatible, apart from shaders, as Optifine uses a different graphics API (OpenGL), whilst Re-Fine uses Vulkan
-  Audio enhancements, such as better 3d audio, realtime audio physics, more SFX (sounds)
-  Visual enhancements, such as vulkan based shaders, particle physics, more VFX (paticles)
-  More will be added to this list.
